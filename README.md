# Calculadora-em-HTML-CSS-e-JavaScript
Calculadora simples desenvolvida com HTML, CSS e JavaScript, contendo as quatro operações matemáticas básicas (adição, subtração, multiplicação e divisão), além de um botão para limpar os cálculos. Possui interface intuitiva, design responsivo e lógica implementada em JS para realizar as operações de forma dinâmica e eficiente. #HTML #CSS #JS.
